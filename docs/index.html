arXiv · ●●●○○ Significant
›

Researchers connect diffusion models to scale-space theory, showing that noisy diffusion states contain no more information than small, downsampled images. They propose Scale Space Diffusion, a framework that fuses scale spaces into the diffusion process using downsampling as degradation.
