InverFill Improves Image Inpainting
arXiv · ●●○○○ Noteworthy
›

InverFill is a one-step inversion method for image inpainting that injects semantic information into initial noise, enabling high-fidelity few-step inpainting. It leverages few-step text-to-image models to improve harmonization and reduce artifacts between the background and inpainted region.

InverFill's approach to image inpainting can improve the efficiency and quality of image generation and editing tasks, particularly in applications where speed and fidelity are crucial.

LLMs Struggle With Code Evolution
arXiv · ●●○○○ Noteworthy
›

Researchers evaluated large language models for automated unit test generation under software evolution, finding that while LLMs achieve strong baseline results, their performance degrades as programs evolve. The study analyzed eight LLMs and 22,374 program variants, revealing weaknesses in LLM-generated tests.

This research matters because it highlights the limitations of LLMs in generating robust tests that can adapt to changing codebases, which is crucial for ensuring the reliability and maintainability of software systems at scale.
