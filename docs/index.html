This study analyzes token consumption patterns in agentic coding tasks, finding that these tasks are uniquely expensive and token usage is highly variable. The research evaluates eight LLMs on their ability to predict their own token costs before task execution, with results showing that higher token usage does not translate into higher performance.

Understanding token consumption patterns in agentic coding tasks is crucial for optimizing the efficiency and cost-effectiveness of AI agents in complex workflows.
