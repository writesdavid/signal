Weight Tying Favors Output
arXiv · ●●●○○ Significant
›

Weight tying in language models biases token embeddings towards output space, rather than input representation. This bias arises from dominant output gradients early in training, affecting early-layer computations.

This matters for language model design, as it highlights a trade-off between input representation and output prediction when using weight tying.
